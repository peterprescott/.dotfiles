We use i3 as the Linux window manager.
Here's how it works.

The key to the left of your ALT key is 
called the SUPER key. (It might have a
Windows logo on it).

- SUPER+ENTER opens a new terminal
- SUPER+g opens the web-browser
- SUPER+y/u/i/o/p moves you to workspace 1/2/3/4/5
- SUPER+d opens the program launcher at the top
- SUPER+x closes the GUI (press x again to confirm) 
- SUPER+q closes the window
- SUPER+f makes the window full-screen
- SUPER+a allows you to resize the window (with h/j/k/l)
- SUPER+w displays this welcome message again

To see and edit the full configuration, 
type `i3config` in the terminal.
